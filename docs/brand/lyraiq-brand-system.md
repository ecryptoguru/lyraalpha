# LYRAIQ Brand System v1

## Core logo assets
- `public/brand/lyraiq-symbol-v1.svg`
- `public/brand/lyraiq-wordmark-v1.svg`
- `public/brand/lyraiq-logo-lockup-v1.svg`

## Concept mapping
- Symbol: abstract constellation/node graph with an embedded subtle `L` path.
- Meaning: intelligence network, multi-asset connections, AI cognition.
- Wordmark: uppercase `LYRAIQ` with slight tracking.
- Feel: calm, intelligent, premium.

## Color system
- Deep Space Blue: `#0B0F1A`
- Electric Indigo: `#5B5FFF`
- Neural Purple: `#8A7CFF`
- Signal Cyan: `#00D4FF`
- Glow gradient: Neural Purple to Signal Cyan.

## Typography
- Heading stack: Satoshi, Space Grotesk, Inter, DM Sans, Arial, sans-serif.
- Body stack: Inter, DM Sans, Arial, sans-serif.
- Style: geometric sans-serif, high legibility, no decorative fonts.

## Motion language
- Use slow, intentional transitions (`600ms` to `1200ms`, `ease-in-out`).
- Animate connecting lines with low-amplitude drift.
- Nodes should pulse softly (subtle scale and glow, never flashy).

## Surface language
- Dark-first surfaces using `#0B0F1A` as base.
- Glassmorphism layers with low alpha and soft blur.
- Orbital curves, network graphs, and layered grid accents.

# LyraAlpha 90-Day Launch Marketing Pack

This folder contains the step-by-step launch plan for reaching 10,000 users in 90 days.

## Assumptions

- Goal: 10,000 signups/users in 90 days
- Geo focus: India + US
- Launch motion: organic-first, supported by partnerships, PR, referral loops, and selective paid retargeting
- Primary product angle: fintech SaaS / financial intelligence / AI market copilot

## Files

- `01-week-by-week-90-day-calendar.md` — execution plan by week
- `02-blog-content-calendar.md` — exact blog titles and publishing cadence
- `03-social-posting-calendar.md` — Twitter/X and LinkedIn calendar
- `04-kpi-dashboard-and-growth-funnel-model.md` — KPI dashboard and growth funnel model
- `05-pr-and-partnership-outreach-list.md` — outreach targets and messaging framework
- `06-aggressive-daily-action-plan.md` — day-by-day 90-day execution plan for a faster user ramp
- `07-india-first-vs-us-first-growth-playbook.md` — regional strategy variants for India-first vs US-first launch motion
- `50-SEO-IN.md` — India retail investor keyword map for SEO/GEO
- `50-SEO-US.md` — US retail investor keyword map for SEO/GEO
- `100-SEO-pages.md` — 100-page tool and answer-page roadmap

## How to use this pack

- Start with the weekly plan to understand the overall 90-day arc.
- Use the blog calendar to populate the editorial pipeline.
- Use the social calendar to drive daily distribution.
- Use the KPI dashboard model to measure weekly performance.
- Use the PR and partnership list to build credibility and distribution.
- Use the aggressive daily plan if you want to execute at a faster cadence.
- Use the regional playbook to localize the launch for India-first or US-first growth.
- Use the India/US SEO keyword docs to route search intent into blogs, tools, and comparison pages.
- Use the 100-page SEO roadmap to turn the keyword maps into page-level execution.

## Launch principle

The launch should compound from a simple loop:

1. Publish useful insight.
2. Distribute it everywhere relevant.
3. Capture the attention with a clear CTA.
4. Activate the user quickly.
5. Retain them with ongoing value.
6. Turn the best users and partners into distribution.

# GEO Optimization Checklist (Per Page)

## Structure for AI Extraction:
- [ ] **H1:** Contains primary keyword, clear and direct
- [ ] **First paragraph:** Direct answer to the query (30-40 words)
- [ ] **H2 sections:** Address fan-out sub-queries (3-5 sections)
- [ ] **Bullet points:** Use for scannable lists (30-40% higher AI visibility)
- [ ] **Numbered lists:** Use for step-by-step processes
- [ ] **FAQ section:** 3-5 Q&As targeting related questions

## E-E-A-T Signals:
- [ ] **Expert quotes:** Include with attribution ("According to [Name], [Title]")
- [ ] **Cited statistics:** Show sources (CoinGecko, DeFiLlama, etc.)
- [ ] **First-hand experience:** Case studies, tool usage, test results
- [ ] **Author information:** Clear byline with credentials
- [ ] **Last updated date:** Show freshness

## Schema Markup:
- [ ] **Article schema:** For blog content
- [ ] **FAQ schema:** For question pages
- [ ] **Tool schema:** For interactive tools
- [ ] **HowTo schema:** For guides

## Content Freshness Plan:
- [ ] **Quarterly updates:** Top 20 pages refreshed every 3 months
- [ ] **Monthly data refresh:** Prices, yields, metrics
- [ ] **Weekly narrative updates:** Market trends, hot topics

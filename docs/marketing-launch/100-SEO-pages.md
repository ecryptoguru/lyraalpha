To drive **mass organic traffic**, **InsightAlpha AI** should build **SEO pages that answer real investor questions and provide tools**, not just articles.

The best strategy is to build **three types of pages**:

1. **AI answer pages (question-based)**
2. **stock / asset intelligence pages**
3. **interactive investing tools**

Below is a **100-page SEO roadmap** designed to capture **hundreds of thousands of monthly searches from India + US investors**.

## Keyword map source of truth

Use the regional keyword documents as the input layer for every page decision:

- `50-SEO-IN.md` for India retail investor search intent
- `50-SEO-US.md` for US retail investor search intent

This means every page in the roadmap should answer one of these three questions:

- Which keyword cluster does this page target?
- Is this page better for India, US, or both?
- Does this page work best as an article, a tool, or a question-based answer page?
